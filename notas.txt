Hoy aprendi a usar el comando git clone, que es para clonar un repositorio que esta en la nube y poder editarlo.
Además, aprendia la importancia del archivo .ignore para poder resguardar nuestras credenciales o datos sensibles al momento de hacer un pull.
Finalmente, aprendí la importancia del comando git diff para ver las diferencias en los cambios locales y el staging antes de subirlo, y que podemos usarlo cuando no estamos usando un entorno, como VSC.
